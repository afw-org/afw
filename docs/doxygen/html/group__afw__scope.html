Create an Adaptive Framework scope.

Parameters
nameof scope.
numberthat is unique within the life of this Adaptive Framework application for this scope name.
scopeof caller. This will be scope's parent.

Call each time a new scope is needed with a shorter lifetime. For example, a server scope with a shorter lifetime request_session scope. Calling the release method will release all resources allocated in the scope including any children scopes. Call afw_environment_create() to create the initial scope.

The scope authorization mode is set to user. Use AFW_SCOPE_AUTHORIZATION_MODE_BEGIN/END to change it.

The "get" function is the fastest way to get the value of a known variable for this pushed qualifier. The qualifier_object is used when the get_next_property() method is needed.
