Adaptive Function to_string<objectPath>

Parameters
functionstruct for this function.
argcnumber of values in argv.
argvlist of values.
pto use for result.
xctxof caller.

Converts objectPath value to string. For list values, the to_string() value for each entry is returned separated with commas.

This function is pure, so it will always return the same result given exactly the same parameters and has no side effects.

Declaration:

function to_string<objectPath>( value: objectPath ): string;

Parameters:

value - (objectPath) A objectPath value.

Returns:

(string) The string representation of the value.
