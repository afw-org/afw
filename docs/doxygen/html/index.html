What is Adaptive Framework?

Adaptive Framework is a set of interfaces for developing servers, requests, object access, etc. in an adaptive way, along with implementations, language bindings, commands, and administrative applications.

Source Code

The source for Adaptive Framework is located in a GitHub repository.

Development is currently being done in the develop branch, but will be merged into main once it's in a state appropriate for public use.
